# 競艇分析アシスタント

本命モード(①定性分析 / ②AI+シミュレーション統合 / ③シミュレーションのみ)と
穴狙いモード(市場が過小評価している艇・組み合わせを探す)の2タブ構成。

## セットアップ

```bash
npm install
```

`.env.local` を作成し、Anthropic APIキーを設定してください(.env.exampleを参照)。

```
ANTHROPIC_API_KEY=sk-ant-...
```

## ローカル起動

```bash
npm run dev
```

http://localhost:3000 で確認できます。

## Vercelへのデプロイ

1. このリポジトリをGitHubにpush
2. Vercelで「New Project」→ このリポジトリを選択
3. Environment Variablesに `ANTHROPIC_API_KEY` を設定(Production/Preview両方)
4. Deploy

APIキーはサーバー側(API Routes)でのみ使用され、ブラウザには公開されません。

## 主な機能

- 画像アップロード(1枚〜複数枚、出走表・直前情報・展示・オッズなど種類自由)
- 本命モード:
  - ①画像から定性分析(展示・整備変化・当地適性・フライング解釈などを考慮)
  - ②①+5万回モンテカルロシミュレーションを統合した最終判断
  - ③シミュレーションのみ(期待値トップ5・推奨買い目・3周の想定展開図)
  - 見送り推奨バッジ(全combo期待値マイナス、または大混戦時)
  - 直前オッズ画像(1枚でも可)で「前回」を引き継ぎ、オッズ変化表→結論の形式に
- 穴狙いモード:
  - 荒れ度判定、市場の過小評価艇の抽出、穴候補フォーメーション提示
  - 固定の会場テーブルは使わず、毎回そのレースのデータから判断
- 競艇場・レース番号は検索付きドロップダウン(候補から選択 or 自由入力)
- 結果ログ・調整メモ(ブラウザのlocalStorageに保存、次回分析時に反映)
